# Summer School DMDU

Preparation course for the summer school DMDU 2022.

## Content
* Variables and data types.
* Collections:
	- Lists
	- Tuples
	- Sets
	- Dictionaries
* Control Flow:
	- while loop
	- if-elif-else
	- for loop
* List and Dictionaries comprehensions.
* Functional programming.
* Functions:
	- Basic functions definitions.
	- Lambda functions.
* Reading and writing files.
* Numpy
* Pandas
